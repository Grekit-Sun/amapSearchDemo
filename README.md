
本工程为基于高德地图Android SDK进行封装，实现了通过地图中心进行兴趣点选择的例子
## 前述 ##
- [高德官网申请Key](http://lbs.amap.com/dev/#/).
- 阅读[开发指南](http://lbs.amap.com/api/android-sdk/summary/).
- 工程基于Android 3D地图SDK和搜索SDK实现

## 功能描述 ##
基于3D地图SDK和搜索SDK进行封装，通过屏幕中心点经纬度进行逆地理编码搜索和POI搜索。
